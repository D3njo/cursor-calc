# Cursor Calc ⬡

A single-file PWA to estimate [Cursor AI](https://cursor.sh) Composer session costs
across different models (Claude, GPT-5.5, etc.).

## Features

- **Workflow Steps** – configure input/output tokens per step (Review, Code, ...)
- **Session Config** – set N sessions + main model
- **Review Session** – separate model + token budget for code reviews
- **Model Editor** – add/edit/remove models with custom pricing ($/M tokens)
- **Results** – cost breakdown, step-level detail, plan coverage (Pro / Pro+ / Ultra)
- **PWA** – installable on mobile & desktop, works offline
